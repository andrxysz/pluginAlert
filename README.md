# Plugin de Alertas – Minecraft 1.8.9

Este plugin adiciona um sistema completo de alertas para servidores Minecraft 1.8.9.
É possível utilizar com BungeeCord para enviar alertas em determinado servidor.

## Permissões úteis:

```bash
alert.use
op
```

## Comandos
```bash
/alert here (mensagem)
/alert (servidor) (mensagem)
```
